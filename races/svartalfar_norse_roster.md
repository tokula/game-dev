# Svartálfar — Norse Roster

> See also: [svartalfar_norse.md](svartalfar_norse.md)

## Command hierarchy

No named Svartálfar rulers or hierarchy are attested in primary sources as distinct from the Dvergar. When the Prose Edda describes visits to Svartálfheimr, the beings encountered are identified as dwarves.

**Source priority:** Primary sources (Prose Edda) are authoritative.

## Named individuals — Primary sources

No individual Svartálfar are named in primary Norse sources as a category distinct from Dvergar.

> **Note:** If Svartálfar are equated with Dvergar — the position held by many scholars — see [dvergar_norse_roster.md](dvergar_norse_roster.md) for the full named roster. If treated as a separate race, the primary record provides no named individuals.
